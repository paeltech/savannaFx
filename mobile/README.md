# SavannaFX Mobile App

Native mobile application for SavannaFX built with Expo and React Native.

## Setup

1. Install dependencies:
```bash
npm install
```

2. Start the development server:
```bash
npm start
```

3. Run on iOS:
```bash
npm run ios
```

4. Run on Android:
```bash
npm run android
```

## Project Structure

- `app/` - Expo Router app directory with file-based routing
- `components/` - Reusable React Native components
- `lib/` - Utility functions and configurations (Supabase client)
- `constants/` - App constants (Colors, etc.)
- `hooks/` - Custom React hooks
- `types/` - TypeScript type definitions

## Features Implemented

- ✅ Project foundation with Expo Router
- ✅ NativeWind (Tailwind CSS) configuration
- ✅ Supabase client setup with secure storage
- ✅ Home/Dashboard screen matching design
